Release 35

I've added entity resolution to the XML classes.

What's New

CRandomNumberGenerator2 - Thanks go to Yehuda Hahn (ygh@cfsnet.com), for turning me on to a fast random number generator by using Mersenne Twister algorithms developed by Makoto Matsutomo and Takuji Nishimura of the Department of Mathematics, Keio University, in Japan (at http://www.math.keio.ac.jp/matumoto/emt.html).

Bug Fixes/Enhancements

CNetwork - Fixed a bug, thanks to Ullrich Pollahne (u.pollaehne@online.de), where I was not allocating room for any stripped leading backslashes in a server name in the Open() method.

CRemoteAccessService - Fixed a bug, thanks to Richard Wirth (r.wirth@wirthware.da-net.de), where the Open() method could go into an endless loop.

CPing - Fixed a bug, thanks to Curits Yanko (cyanko@ziplink.net), where I was not thunking the call to inet_addr().

CRegistry - Fixed a bug, thanks to George Koukoulommatis (gemasoft@hol.gr), where I was not calculating the size of a string buffer correctly. I was incorrectly assuming that a character was only one byte wide which is not UNICODE correct. This bug was in the SetStringArrayValue() function.

CSimpleSocket - Fixed a bug in SetAddress() where the name and address properties weren't being set if there was no DNS entry for an IP address.

Map2Def - This is a sample utility that converts a MAP file to a DEF file. I fixed it where it was not including the timestamp in the resulting DEF file.

XMLCheck - This is a new sample utility that loads an XML file and tells you if it is valid or not. If it is not a valid file it will tell you exactly where it failed (line number and column number).
