 This Release

This release fixed some serious linker problems with the STL builds (little programming hint, don't try to build a library with inline functions that use STL things in a header file). Other than that, there's minor bug fixes and documentation changes. Updates may come more frequently as I begin work on Windows 2000 (aka W2K, NT5). W2K is amazingly complex so I will have plenty of API's to encapsulate.
